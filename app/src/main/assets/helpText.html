Wave~Steg:

Write: Write encrypted messages to a WAV file.

  • Password: Select a password with which the message is to be encrypted and then decrypted again later.
  • Message: The message to hide in the WAV file
  • File: The WAV file in which the encrypted message should be hidden

Read: Read an encrypted message from a WAV file.

  • Password: The password you used to encrypt the message in the WAV file, or that you were told. If the password is wrong, you will receive a wrong decrypted message
  •  File: The WAV file from which you want to try to read a message.
